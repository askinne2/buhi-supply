# Static images

Add these assets for the site to match the design:

- **hero.jpg** – Hero section background (recommended: 1440×800 or similar)
- **lifestyle/** – Work, School, Travel, Gym category images
- **products/** – Product thumbnails
- **ugc/** – UGC grid images for #BuhiEverywhere

Until `hero.jpg` exists, the Hero section will need a fallback or placeholder.
